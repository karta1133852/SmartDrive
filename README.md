# SmartDrive
## NCKU CSIE 大三專題
- 這是一款可以提供駕駛者前方路口號誌的APP，透過讀取使用者的位置、移動方向與移動速度，判斷目前行駛在哪條路上，並提供下個路口的號誌時相，經由提醒路口號誌狀態給予使用者更多行車資訊，幫助駕駛者判斷行駛方式(例如應加速或減速)
- 僅實作
  - 行進狀態判讀演算法
  - 秒數計算與倒數動畫
